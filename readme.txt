#Техническое задание
##Введение
В нашем проекте стояла задача сделать так что бы люди желающие поделится своими мыслями, могли выкладывать "Блоги"
##Для чего этого нужно
Для создания удобства пользователям, как создающим блоги так и учавствующим в них.
##Цели
1.Стояла задача сделать так что бы главная страница блога содержала превью постов автора, отсортированных по дате публикации.
2.Возможность администрирования, что бы у блога был администратор, который и является владельцем блога.
3.Что бы администратор через админ. панель мог:

	#добавлять/удалять/редактировать посты
	#добовлять/удалять комментарии
	#удалять пользователей

4.Создание возможности авторизоваться под "Блогом", ввести логин – admin и пароль – 12345

#Для пользователей
1.Добавить возможность быстрой регистрации, для участия в "Блоге"
2.Возможность пользователям оставлять комментарии к постам "Блога"
3.Что бы у авторизованного пользователя при просмотре поста полностью, после поста появилась форма отправки комментария к посту

#Создать списки зарегестрированных пользователей например:
 Student:
 Петя
 Вася
 John
 Алексей
Пароли у всех – 12345
Пароли должны хешироватся при хранении в БД

#Запуск проекта: Для запуска пиложения необходимо установить все модули, указаные в файле requirements.txt
 Для того, чтобы запустить приложение, необходимо запустить файл main.py с помощью среды разработки pyCharm или любым другим способом. 
 Web-приложение запустится по адресу: http://127.0.0.1:8080/ .
 Для того, что бы поменять хост и порт на которых работает web-блог, нужно в ручную поменять эти параметры при вызове метода run в файле main.py в самом конце кода
